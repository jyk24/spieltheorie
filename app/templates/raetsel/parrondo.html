🪙 Parrondo-Paradoxon

Zwei Verlierer-Spiele werden zusammen zum Gewinner.

Die Behauptung:

Spiel A (Verlust)  +  Spiel B (Verlust)  =  Gewinn

Juan Parrondo entdeckte 1996 ein verblüffendes Ergebnis: zwei Glücksspiele mit jeweils negativem Erwartungswert können – wenn man zwischen ihnen abwechselt – einen positiven Erwartungswert haben. Das widerspricht der Intuition: Verlust + Verlust sollte doch erst recht Verlust sein.

Die zwei Spiele:

Spiel A Eine leicht unfaire Münze
P(Gewinn +1€) = ½ − ε  ·  P(Verlust −1€) = ½ + ε

Erwartungswert pro Runde: −2ε → langfristig Verlust.

Spiel B Hängt vom aktuellen Kapital ab
Kapital % 3 == 0: P(Gewinn) = 1/10 − ε  („Pech-Münze")
sonst:             P(Gewinn) = 3/4 − ε  („Glücks-Münze")

Auch hier: Erwartungswert < 0 für ε > 0 (schon bewiesen).

Warum funktioniert das?

Spiel B nutzt die Pech-Münze immer dann, wenn das Kapital ein Vielfaches von 3 ist. Bei dauerhaftem Spiel B verbringst du verhältnismäßig viel Zeit im Pech-Zustand und verlierst.

Schaltet man Spiel A dazwischen, „mischt" man die Kapitalverteilung. Der Pech-Zustand wird seltener besucht. Die Glücks-Münze von Spiel B kommt häufiger zum Einsatz – und kippt die Bilanz.

Brownsche Mühle als Inspiration

Parrondo entwickelte das Paradoxon nicht aus reinem Glücksspielinteresse, sondern um die Brownsche Ratsche zu erklären: Feynman hatte gezeigt, dass eine Maschine aus reinem thermischem Rauschen keine nutzbare Arbeit gewinnen kann. Parrondo zeigte: Schaltet man asymmetrische Potentiale geschickt um, kann ein Teilchen durchaus „bergauf" wandern – ohne den 2. Hauptsatz zu verletzen.

Wo das Paradoxon wirklich auftaucht

  • Biologie: Mischpopulationen mit eigentlich unterlegenen Strategien können dominieren.
  • Finanzmärkte: Portfolio-Rebalancing zwischen zwei „schlechten" Anlagen kann Rendite erzeugen (Volatility Pumping, Shannon 1960).
  • Genetik: Zwei nachteilige Mutationen können kombiniert vorteilhaft sein.
  • Quantencomputing: Quantenanaloga wurden 2002 demonstriert.

Achtung: kein Trick gegen Casinos

Das Paradoxon braucht eine Korrelation zwischen Spielzuständen (Spiel B kennt dein Kapital). In einem normalen Casino sind Runden unabhängig – dort hilft Wechseln nicht. Roulette bleibt mathematisch hoffnungslos.
