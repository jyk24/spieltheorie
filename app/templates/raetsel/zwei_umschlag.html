✉️ Zwei-Umschlag-Problem

Tauschen oder behalten? Das Argument sagt immer tauschen – aber das kann nicht stimmen.

Die Situation:

✉️ Du bekommst einen von zwei Umschlägen. Du öffnest ihn und siehst {{ amount }} €.
✉️ Der andere Umschlag enthält entweder {{ amount * 2 }} € (doppelt) oder {{ (amount / 2) | int }} € (halb so viel).

Das Argument fürs Tauschen:

Mit 50 % Wahrscheinlichkeit verdoppelst du: {{ amount }} → {{ amount * 2 }} (+{{ amount }}€)

Mit 50 % Wahrscheinlichkeit halbierst du: {{ amount }} → {{ (amount / 2) | int }} (-{{ (amount / 2) | int }}€)

Erwartungswert beim Tausch: 0,5 × {{ amount * 2 }} + 0,5 × {{ (amount / 2) | int }} = {{ (amount * 2 * 0.5 + (amount / 2) * 0.5) | round(1) }}€ > {{ amount }}€

→ Also: immer tauschen?

Deine Entscheidung:

Du hast {{ amount }} € im Umschlag. Was machst du?
