🛣️ Braess-Paradoxon

Eine neue Straße kann alle verlangsamen – Nash-Gleichgewicht vs. gesellschaftliches Optimum.

Das Netzwerk:

4000 Pendler fahren täglich von Stadt A nach Stadt B. Es gibt zwei Routen:

Route Oben: A → X → B
A→X: Dauer = t/100 Minuten (t = Anzahl Fahrer auf dieser Strecke)
X→B: Dauer = 45 Minuten (feste Strecke, unabhängig von t)
Route Unten: A → Y → B
A→Y: Dauer = 45 Minuten (feste Strecke)
Y→B: Dauer = t/100 Minuten (variabel)
Ohne neue Straße: Bei 2000 Fahrern auf jeder Route: 2000/100 + 45 = 65 Minuten für alle.

Jetzt wird eine neue Straße gebaut: X → Y (Dauer: 0 Minuten)

Neue Option: A → X → Y → B
Kosten: t/100 (A→X) + 0 (X→Y) + t/100 (Y→B) = t/100 + t/100 Minuten

Die Entscheidung:

Die Stadt überlegt, die neue Straße X→Y zu bauen. Sie kostet nichts – eine "kostenlose" Verbesserung. Sollte die Stadt die Straße bauen?
