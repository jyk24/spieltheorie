😴 Das Schlafendes-Mädchen-Paradoxon

Eine einfache Frage – die die Wahrscheinlichkeitstheorie spaltet.

Das Experiment:

  1. Das schlafende Mädchen wird über das Experiment informiert.
  2. Ein fairer Münzwurf: Kopf oder Zahl.
  3. Bei Kopf: Sie wird einmal aufgeweckt (Montag).
  4. Bei Zahl: Sie wird zweimal aufgeweckt (Montag + Dienstag) – aber ihr Gedächtnis wird zwischen den Aufweckungen gelöscht.
  5. Jedes Mal wenn sie aufwacht, fragt man: „Wie hoch ist die Wahrscheinlichkeit, dass die Münze Kopf zeigte?"
Sie weiß alles über das Experiment. Sie sieht keinen Kalender. Sie weiß nur: sie ist wach.

Was antwortest du als schlafendes Mädchen?

Die Münze wurde geworfen. Du bist aufgewacht. Wie hoch ist P(Kopf)?
