Das mathematische Prinzip: Harmonisches Mittel

✗ Falscher Ansatz (arithmetisches Mittel): (50 + 100) / 2 = 75 km/h → falsch!
✓ Richtig: Durchschnitt = Gesamtweg / Gesamtzeit = 100 km / (50/v₁ + 50/v₂)
v̄ = 2·v₁·v₂ / (v₁ + v₂)
← harmonisches Mittel, nicht arithmetisch!
Umgestellt: v₂ = v̄·v₁ / (2·v₁ − v̄)
{% if impossible %}
Bei v₁={{ v1 }}, v̄={{ v_target }}: 2·{{ v1 }}−{{ v_target }} = {{ 2*v1 - v_target }} ≤ 0 → keine Lösung
{% else %}
v₂ = {{ v_target }}·{{ v1 }} / (2·{{ v1 }}−{{ v_target }}) = {{ v2 }} km/h
{% endif %}

Der unmögliche Spezialfall: v₁ = 50, Ziel = 100

Dies ist der bekannteste Fall. Für 100 km/h Durchschnitt brauchst du insgesamt 1 Stunde für 100 km. Aber für die erste Hälfte hast du bereits 1 Stunde verbraucht (50 km / 50 km/h = 1h). Für die zweite Hälfte bleibt 0 Sekunden – also ∞ km/h.

Allgemein: Das Ziel wird unmöglich, sobald v̄ ≥ 2·v₁. Dann hat man das gesamte Zeitbudget schon in der ersten Hälfte aufgebraucht.

Verhandlungsrelevanz:

Das harmonische Mittel taucht immer auf, wenn man gleiche Mengen zu verschiedenen Raten berechnet: Preise ($/Einheit), Renditen (Zins auf gleiches Kapital), Produktionszeiten. In Verhandlungen: wer Durchschnittswerte falsch berechnet (arithmetisch statt harmonisch), überschätzt systematisch Renditen und unterschätzt Kosten – ein häufiger Fehler in Dealmodellen.

Klassisches Problem der Analysis. Das harmonische Mittel H(a,b) = 2ab/(a+b) wird bei gleichen Strecken- oder Mengenanteilen verwendet. Gegenteil: arithmetisches Mittel bei gleichen Zeitanteilen. Verwechslung dieser beiden ist eine häufige Quelle von Berechnungsfehlern.
