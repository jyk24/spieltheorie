🔮 Argument für nur Box B:

Evidential Decision Theory: Betrachte die Evidenz.

  • → Wenn Vorhersager Recht hat (95 %): Box B = 1.000.000 €
  • → Wenn falsch (5 %): Box B = 0 €
  • → E[nur B] = 0.95 × 1M + 0.05 × 0 = 950.000 €
  • → E[beide] = 0.05 × 1.001M + 0.95 × 1.000 = 51.050 €
⚖️ Argument für beide Boxen:

Causal Decision Theory: Betrachte die Kausalität.

  • → Box B-Inhalt ist bereits festgelegt – meine Wahl ändert ihn nicht.
  • → Falls B = 1M: beide = 1.001M > nur B = 1M ✓
  • → Falls B = 0: beide = 1.000 > nur B = 0 ✓
  • → Beide Boxen dominiert "nur B" in jedem Zustand der Welt.

Wo liegt der Widerspruch?

Beide Argumente sind formal korrekt – und führen zur gegenteiligen Handlung. Das ist das Paradoxon. Es spaltet die Entscheidungstheoretiker seit 1960:

  • → Robert Nozick (1969) formalisierte das Problem – und wechselte mehrfach die Meinung.
  • → Umfragen: ~55 % der befragten Philosophen wählen nur B (Evidential), ~45 % beide (Causal).
  • → Kernfrage: Kann eine Entscheidung Evidenz über eine bereits festgelegte Vergangenheit sein?

Verhandlungsrelevanz:

Das Newcomb-Problem steckt in vielen Commitment-Situationen: Wenn dein Gegenüber dein Verhalten vorhersagen und auf Basis dieser Vorhersage handeln kann, bevor du handelst – wie rational sollst du dann handeln? Dominanzargumente und Korrelations-Kausalitäts-Unterscheidungen sind in strategischen Entscheidungen allgegenwärtig.
